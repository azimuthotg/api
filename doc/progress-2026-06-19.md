# Progress 2026-06-19 — Monitor: ติดตามวันหมดอายุ JWT token

## ที่มา (ปัญหาจริง)
API ออก JWT ให้ระบบปลายทางใช้ (access อายุ **365 วัน**) แต่เป็น stateless — ไม่มี record
ว่าใครได้ token ไปเมื่อไหร่/หมดเมื่อไหร่ เมื่อวาน (2026-06-18) **2 ระบบล่ม**เพราะ token หมดอายุ
ใช้เวลานานกว่าจะรู้สาเหตุว่าเป็นเพราะ token หมด

## สิ่งที่ทำ (2 ฟีเจอร์ใน /monitor/)

### 1. Token Inspector — `/monitor/token/`
วาง JWT (access/refresh) → ตรวจ signature ด้วย `SECRET_KEY` (HS256) แต่ **ไม่เช็ค exp**
(จึงดู token ที่หมดไปแล้วได้) → แสดง token_type / user_id / iat / exp / jti / เหลือกี่วัน
+ แถบสถานะ หมดแล้ว(แดง)/ใกล้หมด ≤30 วัน(เหลือง)/ปกติ(เขียว). ใช้วินิจฉัยเร็วเวลาระบบล่ม.

### 2. Token issuance log + dashboard — `/monitor/tokens/`
- model `TokenIssueLog` (table `token_issue_log`, migration `0006`) — event(obtain/refresh),
  username, user_id, jti, issued_at, expires_at, ip, user_agent.
- ครอบ endpoint ออก token: `apiapp/token_views.py` →
  `LoggingTokenObtainPairView` / `LoggingTokenRefreshView` (subclass ของ simplejwt,
  log หลังออกสำเร็จ, ถ้า log พังไม่กระทบ auth). wire ใน `apiproject/urls.py` แทนของเดิม
  (URL/พฤติกรรมเดิมทุกอย่าง).
- dashboard สรุป **token ล่าสุดของแต่ละ user** เรียงใกล้หมดก่อน + ไฮไลต์สี
  + การ์ดสรุป (จำนวน user, หมดแล้ว, ใกล้หมด ≤30 วัน) + ประวัติการออกทั้งหมด (ค้น username + แบ่งหน้า).

## โครงสร้างไฟล์
- `apiapp/token_utils.py` — `decode_token()` (PyJWT, verify sig ไม่ verify exp), `log_token_issue()`.
- `apiapp/token_views.py` — logging token views + `_username_from_access()`.
- `apiapp/monitor_views.py` — `monitor_token_inspect`, `monitor_token_log`, `_token_status()` (เกณฑ์ `TOKEN_SOON_DAYS=30`).
- templates `monitor/token_inspect.html`, `monitor/token_log.html`. ปุ่มลิงก์เพิ่มใน dashboard เดิม.
- `apiapp/admin.py` — register `TokenIssueLog`.

## ข้อจำกัด (สำคัญ)
log เริ่มเก็บ**เฉพาะ token ที่ออกหลังจาก deploy นี้** — ระบบที่ถือ token เก่าอยู่แล้ว
(ออกก่อนหน้า) จะยังไม่โผล่ใน dashboard จนกว่าจะ re-login. ช่องว่างนี้ใช้ **Token Inspector**
อุดได้ทันที (เอา token ปัจจุบันของระบบนั้นมาวางดูวันหมด).

## เทสต์ที่ทำ (local, ยังไม่แตะ prod DB)
